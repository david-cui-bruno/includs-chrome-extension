# Extension Icons

Add the following icon files to this directory:

- `icon16.png` (16x16 pixels)
- `icon48.png` (48x48 pixels)
- `icon128.png` (128x128 pixels)

For testing, you can use any square PNG images. The extension will still load without icons, but Chrome will show a default placeholder.

## Quick Placeholder Icons

You can generate simple placeholder icons using any image editor or online tool like:
- https://favicon.io/
- https://realfavicongenerator.net/

Or use a simple colored square as a placeholder.
